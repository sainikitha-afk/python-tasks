# Python Tasks 

## Overview

This repository contains a collection of advanced Python projects covering backend development, distributed systems, data engineering, system design, and core computer science concepts.

Each task is designed to simulate real-world engineering problems and demonstrate practical implementations of scalable architectures, asynchronous systems, and data processing pipelines.

---

## Tech Stack

* Python 3.x
* Async Programming (`asyncio`, `aiohttp`, WebSockets)
* Data Processing (`pandas`, `numpy`)
* Backend Frameworks (`FastAPI`)
* Databases (SQLite, in-memory stores)
* Distributed Systems Concepts
* Machine Learning (`scikit-learn`)
* Visualization (`matplotlib`)
* PDF Generation (`ReportLab`)

---

## Tasks Summary

### Task 1: Web Scraper with Pagination

A CLI-based web scraper that extracts structured data from paginated websites and stores results in JSON format.

### Task 2: Real-Time Chat Application

A WebSocket-based chat server supporting multiple users, real-time messaging, and interactive communication.

### Task 3: Custom ORM

A lightweight Object-Relational Mapper built using metaclasses and descriptors with support for CRUD operations and query chaining.

### Task 4: Distributed Task Queue

A producer-consumer system using Redis that supports retries, task distribution, and result tracking.

### Task 5: Machine Learning Pipeline

An end-to-end ML pipeline with feature engineering, model training, evaluation, and model persistence.

### Task 6: API Gateway with Rate Limiting

An async API gateway implementing caching, rate limiting, and circuit breaker patterns.

### Task 7: Mini Language Interpreter

A custom interpreter with lexer, parser, AST, and execution engine supporting functions and recursion.

### Task 8: Real-Time Data Streaming Dashboard

A streaming system that processes live data and visualizes it in real-time using WebSockets.

### Task 9: Concurrent Web Crawler

An asynchronous crawler with BFS traversal, depth control, and sitemap generation.

### Task 10: Blockchain Prototype

A simplified blockchain implementation with proof-of-work, transactions, and wallet system.

### Task 11: Automated Testing Framework

A mini testing framework with decorators, test discovery, and execution reporting.

### Task 12: Graph Database Engine

An in-memory graph database supporting traversal queries, shortest path, and persistence.

### Task 13: PDF Report Generator

A system that generates structured PDF reports using templates, charts, and conditional sections.

### Task 14: Concurrent Web Crawler (Advanced)

An enhanced crawler with concurrency, robots.txt compliance, SEO auditing, and graph export.

### Task 15: Event-Driven Microservice (CQRS)

A microservice architecture using CQRS and event sourcing with async event handlers and read models.
